### Fantasktic Component Library

To install:

`yarn add @fantaskticedtechlimited/fui-complib`

To update:

`yarn remove @fantaskticedtechlimited/fui-complib`
`yarn add @fantaskticedtechlimited/fui-complib`

or 
`yarn add @fantaskticedtechlimited/fui-complib@`**_X.X.X_**

**_X.X.X_**: version number
