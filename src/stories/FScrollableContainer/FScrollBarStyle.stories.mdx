# FScrollBarStyle

`FScrollBarStyle` is **NOT** a component, but a style function which can be applied on `className` of a component
to show a scrollbar with default or custom style for the overflow content.

Usage (simple, with other classnames):

`<div className={`_`classname`_`+ " " + FScrollBarStyle()}>`_`Content`_`</div>`

Props: `FScrollBarStyleProps`.
