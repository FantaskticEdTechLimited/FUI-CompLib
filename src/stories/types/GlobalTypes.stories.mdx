# Global Types

<h6>The following types are commonly used in types of components.</h6>

<br />

`Nullable<T>`

```tsx
T | null;
```

`NumericStringType`

```tsx
number | string;
```

`FOnClickFunction`

```tsx
() => void
```

`FOnSubmitFunction<T>`

```tsx
((data: T) => void) | ((data: T) => Promise<void>)
```

`FOnDataCallbackFunction<T, C>`

```tsx
(data: T, callbackFunction: C) => Promise<void>
```

`FOnCallBackFuntion<C>`

```tsx
(callbackFunction: C) => Promise<void>
```

`FOnHoverProperty<T>`

```tsx
(isHover?: boolean, index?: NumericStringType) => T;
```

`FComponentType`

```tsx
ReactElement<FIconProps | FSVGIconProps> | ReactNode
```

`FComponentsType`

```tsx
Array<FComponentType>
```
